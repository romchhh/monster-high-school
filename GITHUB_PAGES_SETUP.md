# Налаштування GitHub Pages

## Крок 1: Відкрийте налаштування репозиторію

1. Перейдіть на https://github.com/romchhh/monster-high-school
2. Натисніть на вкладку **Settings** (Налаштування) вгорі репозиторію
3. У лівому меню знайдіть розділ **Pages** (Сторінки)

## Крок 2: Налаштуйте GitHub Pages

1. В розділі **Source** (Джерело) виберіть:
   - **Branch: main**
   - **Folder: / (root)**
2. Натисніть **Save** (Зберегти)

## Крок 3: Очікуйте активації

- GitHub опрацює ваш сайт протягом 1-2 хвилин
- Коли все готово, ви побачите повідомлення: "Your site is live at..."
- Ваш сайт буде доступний за адресою: **https://romchhh.github.io/monster-high-school/**

## Важливо!

Якщо ваші файли не відображаються:
- Перевірте, що всі файли використовують відносні шляхи (як у вашому коді)
- Переконайтеся, що `index.html` знаходиться в корені репозиторію
- Можливо, потрібно зачекати ще кілька хвилин для оновлення

## Оновлення сайту

Кожен раз, коли ви робите `git push`, сайт автоматично оновлюється!

```bash
git add .
git commit -m "Оновлення сайту"
git push
```
