# Monster High School - Лабораторна робота

Сайт про школу Monster High з 5 сторінками та демонстрацією обробки растрових зображень.

## Структура сайту

- **index.html** - Головна сторінка з інформацією про персонажів
- **personazhi.html** - Таблиця персонажів (4×6)
- **galereya.html** - Галерея з 6 зображеннями різного розміру
- **shkola.html** - Статті про школу з зображеннями
- **kontakti.html** - Форма зворотного зв'язку
- **obrobka.html** - Демонстрація обробки растрових зображень (7 варіантів)

## Обробка зображень

Скрипт `process_images.py` створює 7 варіантів оптимізації зображень:
1. 200×200px, 4 кольори, без прозорості
2. 200×200px, 9 кольорів + прозорість
3. 1920×900px, 2 кольори
4. 200×200px, повна якість + прозорість
5. 200×200px, збереження графічних ліній
6. 200×200px, низька якість (JPEG)
7. 200×200px, чорно-біле з растровою точкою
